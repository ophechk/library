Fonctionnalité du site Library 

- un utilisateur non connecté doit s'inscrire ou se connecter pour accéder au contenu du site,
il peut envoyer un message via le formulaire de contact

- un utilisateur connecté peut voir les livres et les détails de ces derniers

- il peut laisser un commentaire (un seul sur chaque livre)

- il peut modifier son profil utilisateur (mot de passe et pseudo)

- il a accès à la galerie de couverture des livres

- il peut se déconnecter également
